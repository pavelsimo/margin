# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.2.0] - 2026-07-21

### Added

- connect OpenAI-compatible providers with model discovery, encrypted credentials, streaming responses, and vision input
- browse PDFs from their embedded document outline and jump directly to bookmarked pages
- copy completed assistant answers and see when each answer was created

### Changed

- improve paper and chat navigation, reader selection and keyboard controls, and background model configuration
- simplify AI provider setup with guided presets and clearer command-line tool detection

## [0.1.0] - 2026-07-19

### Added

- research paper workspace for organizing, reading, and discussing papers with local AI assistants
- named OpenAI-compatible API profiles with model discovery, streaming chat, vision input, and encrypted credentials
